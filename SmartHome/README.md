Third Party Libraries:

- [ZXing.Net for Xamarin.Forms](https://github.com/Redth/ZXing.Net.Mobile) - QR code generation & recognition
- [LifxHttpNet](https://github.com/mensly/LifxHttpNet) - Lifx (wi-fi bulbs) client
